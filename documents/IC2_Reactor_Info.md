# 反应堆

1. 核反应堆:

核电站中只能放燃料棒，散热片，热交换器，反应堆隔板，冷却单元这一类物品

2. 核反应仓:

是为核反应堆提供更大摆放空间的副仓。

每个核反应堆最多增加6个核反应仓，每个核反应仓能够为核反应堆容量增加6格。使用持续红石信号(拉杆、红石火把等)激活核反应仓也可以启动核反应堆。

单个核反应堆本体：3×6 = 18 格

最大总格子数 = 18 + 6×6 = 54 格

当前项目只考虑这54格的情况

# 燃料

1. 铀棒是核反应堆中的燃料。

单铀棒工作时会向四周各发射1束核脉冲（中子流），发电量取决于其收到的核脉冲数，若收不到核脉冲则发电为5EU/t；发热量取决于相邻铀棒或中子反射板数目，发热与铀棒种类（铀棒、双铀棒、四联铀棒）无关。

计算公式：

电量输出=5x (1+核脉冲 ) EU/t

每秒产生热量=2x (n+1) (n+2) ，n=相邻铀棒或中子反射板数目

1热量=1/10,000标准堆温，即一个反应堆在没有反应堆隔板时存储达到10,000热量就会发生爆炸。

铀棒工作时将热量均分给四周可以存储和吸收热量的元件，当周围有3个可储热元件且热量无法均分时，热量余数传给下方的元件（无法传到下方则传到上方），附近没有吸收热量的元件时将会把热量传递给核反应堆，使堆温上升。

# 热交换器

1. 热交换器

热交换器是一种能够交换热量的核反应炉配件。它自身可以储存2500点热量，向相邻其他元件传递12点热量，并1tick向核反应炉自身传递4点热量。

2. 反应堆热交换器：

是一种能够与核反应炉自身交换热量的核反应炉配件。它自身可以储存5000点热量，不会与相邻其他元件交换热量，1tick只会向核反应炉自身传递72点热量。

3. 元件热交换器

它自身可以储存5000点热量，不会与核反应炉自身交换热量，1tick只会向相邻其他元件传递36点热量。

4. 高级热交换器

它自身可以储存10,000点热量，向相邻其他元件交换24点热量，并与核反应堆交换8点热量。

# 散热片

1. 散热片

散热片是一种拥有散发周围燃料棒热量的核反应炉配件。

它自身可以储存1,000点热量并散发自身6点热量。

2. 反应堆散热片

反应堆散热片是一种拥有散热能力的核反应炉配件。它自身可以储存1,000点热量，吸收核反应炉自身5点热量，并散发自身5点热量。

3. 元件散热片

元件散热片是一种拥有散热能力的核反应炉配件。它自身不储存热量，也不吸收任何热量，只会散发周围元件4点的热量。

4. 高级散热片

它自身可以储存1,000点热量，并散发自身12点热量。

5. 超频散热片

它自身可以储存1,000点热量，每tick散发自身20点热量。当堆温不为0时会吸收反应堆热量，速度为36点热量/tick。

这里即使散热堆温，但是仍然不考虑外部控制红石脉冲控制反应堆工作，只考虑稳态工作。

# 中子反射板

1. 中子反射板

中子反射板是一种能够增加各种燃料棒输出频率的元件。中子反射板一共能进行30,000次脉冲，相邻每根单燃料棒每秒消耗1耐久，双联和四联对应消耗2和4耐久。

2. 加厚中子反射板

用于在核反应堆里增加各种燃料棒的输出频率。加厚中子反射板一共能进行120,000次脉冲，相邻每根单燃料棒每秒消耗1耐久，双联和四联对应消耗2和4耐久。

# 冷凝模块

1. 红石冷凝模块

红石冷凝模块是一种能够利用红石来减少核反应炉元件热量的元件，它能够吸收附近元件20,000点热量。使用红石和冷凝模块在工作台上合成可将其修复。每个红石修复冷凝模块10000点耐久。

2. 青金石冷凝模块

它能够吸收附近元件100,000点热量。将青金石冷凝模块与红石在工作台内合成可将其修复。每个红石修复冷凝模块10000点耐久，每个青金石修复冷凝模块40000点耐久。
